# The History of Java ☕

An animated, single-page scroll-story of the Java programming language — from the Green Project and "Oak" in 1991 to virtual threads and Java 25.

One `index.html`, no dependencies, no build step. Pure vanilla HTML / CSS / JS.

## Run locally
Open `index.html` in a browser, or:
```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```
